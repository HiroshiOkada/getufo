# getufo

Download securities reports from EDINET

## Getting Started

```sh
$ npm i -g @toycode/getufo
$ getufo -f <folder to download> <SEC or EDINET code>

```

or

```sh
$ mkdir <some dir>
$ cd <some dir>
$ npm i @toycode/getufo
$ npx getufo -f <folder to download> <SEC or EDINET code>
```

### Prerequisites

"node": ">=8.11"
